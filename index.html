Screenshots

Taking a screenshot on VirtualBox is useful. It provides evidence for doing something, such as getting help on a command or package.

Terminal

Examples of things you can do in Terminal:

  • Clone GitHub Repositories
  • Install or Uninstall Packages
  • View Files
